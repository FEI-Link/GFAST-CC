前端VUE，后端goframe

使用mysql数据库，可以先将这里的例子数据库恢复到自己的数据库里，配置文件在OS/manifest/config里，配置的格式参考goframe框架的规范格式

前端依赖安装 npm install
然后
前端运行 npm run dev

后端运行 go run main.go

现在数据库为可用版本，里面的数据为CHATGPT按要求随机生成
使用gfast前后端结合框架，仅为个人学习调试
